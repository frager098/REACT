# What is JSX?
JSX means Javascript XML. It is basically an HTML like syntax which we can use in JS.
# Why Babel?
1_Babel helps in transpiling modern JS or JS extension into older versions of JS.
2_Makes our development efficient and developer friendly
3_Optimize our code for better performance.

# Why Source Maps?
1_Source Maps map the source file and transpiled file in the browser.
2_Makes debugging easier.
